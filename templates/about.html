About AutoML v1.0

Traditional Machine Learning — Automated, Simplified, Powerful

v1.0.0 Traditional ML Only Flask + Scikit-learn

AutoML v1.0 is an end‑to‑end automated machine learning web platform built entirely from scratch using Python and Flask. Upload a CSV file, let the platform handle data analysis, feature engineering, model training, and performance comparison — all without writing a single line of ML code.

Key Features

Auto EDA + Feature Engineering

Automatic data cleaning, missing value handling, encoding, scaling, and outlier detection.

Multi‑Model Training

Train and compare regression, classification, and zero‑inflated models simultaneously.

Export Everything

Download best model as .pkl, processed dataset as .csv, and full analysis as .json report.

Auto or Manual Mode

Choose fully automated pipelines or manually configure model selection and hyperparameter tuning.

Composite Scoring

Unique Zero‑Inflated composite score for datasets with many zero values — beyond standard metrics.

Live Training Monitor

Real‑time progress tracking with model‑by‑model status updates while training runs in the background.

How It Works

1

Upload Your CSV File

Go to Upload Data and select any CSV file. Index columns and empty columns are automatically removed.

2

Explore Your Data

The Data Overview page shows statistics, column types, missing value counts, and auto‑generated visualizations.

3

Select Target Column

Pick the column you want to predict. The platform auto‑detects regression, classification, or zero‑inflated tasks.

4

Feature Engineering

Click Start Processing to encode categoricals, scale numerics, handle outliers, and prepare data for modeling.

5

Feature Selection

Choose the number of top features to keep, or use Auto mode to retain the most predictive features.

6

Model Training

Choose Auto/Manual mode, set hyperparameters, and start training. Real‑time progress is displayed.

7

Results & Downloads

Compare models side‑by‑side, download the best model, processed CSV, and full JSON report.

Frequently Asked Questions

Currently AutoML v1.0 supports only CSV files. The platform auto‑detects separators (comma, semicolon, tab, pipe). Make sure your file is saved in UTF‑8 encoding for best results.
Zero‑inflated modeling is a two‑stage approach for target columns with a large number of zero values. Stage 1 is a classifier that predicts whether the value will be zero or non‑zero. Stage 2 is a regressor that predicts the actual non‑zero value. AutoML detects this automatically when your target column has >30% zeros.
Training time depends on dataset size, number of models selected, and tuning mode. Auto tuning (RandomizedSearchCV) with many models on large datasets can take several minutes. For faster results, use Manual mode and select 2–3 models with Manual tuning. The progress bar updates in real time.
The .pkl file is a serialized scikit‑learn pipeline. Load it in Python using: 
import joblib
model = joblib.load('your_model.pkl')
predictions = model.predict(your_dataframe)
Make sure to use the processed CSV (not the original) as input, with the same column structure.
Not in v1.0. This version focuses exclusively on traditional machine learning models from scikit‑learn. Deep learning (ANN/CNN/RNN) support is planned for a future version.
Training runs in a background thread on the server, so it continues even if you navigate away. However, we recommend staying on the training page to monitor progress. If you go back and reload the page, the status will reflect the current training state without restarting it.
The JSON report is a comprehensive summary of your entire AutoML session. It includes dataset information, feature engineering steps applied, all model scores and metrics, the best model details, and timestamps. It's useful for documentation and reproducing results.
Yes. Enable Auto Clean on the Target Selection page to automatically fill missing values: numeric columns use median imputation, categorical columns use mode imputation. Rows where the target column itself is missing are always removed.
